aduser.data.examples.maxmind\_geoip module
==========================================

This example shows how you to integrate `MaxMind's GeoIP <https://dev.maxmind.com/geoip/geoip2/geolite2/>`_.

.. NOTE::
    You'll have to install `python-geoip <https://pypi.org/project/python-geoip/>`_. You also need to download the latest database.
