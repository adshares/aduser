aduser.data.examples.browscap module
====================================

This example shows how you to integrate `Browscap <https://browscap.org/>`_.

.. NOTE::
    You'll have to install `this version of pybrowscap <https://github.com/char0n/pybrowscap#egg=pybrowscap>`_. You can downlaod the database using the module.
