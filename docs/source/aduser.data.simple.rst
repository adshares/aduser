aduser.data.simple module
=========================

This data plugin combines the functionality of the following plugins:

`aduser.data.examples.browscap` and `aduser.data.examples.maxmind_geoip`

It's a simple almost out of the box solution for AdUser to provide basic targeting and fraud detection.
